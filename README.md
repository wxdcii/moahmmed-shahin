# Dr. KVNN Santhosh Murthy Portfolio

This project is a high-class, minimalist portfolio for Dr. KVNN Santhosh Murthy, designed to present his biography, achievements, and insights in a unique and engaging manner. The portfolio is built using React and Vite, ensuring a fast and authentic user experience.

## Features

- **Biography Section**: A detailed overview of Dr. Santhosh Murthy's background, education, and professional achievements.
- **Timeline**: A visual representation of significant milestones in Dr. Murthy's career.
- **Blog Section**: A collection of articles and insights authored by Dr. Murthy, showcasing his expertise in the medical field.
- **Responsive Design**: The portfolio is designed to be fully responsive, ensuring a seamless experience across devices.
- **Minimalist Aesthetic**: A clean and modern design that emphasizes content and readability.

## Getting Started

To get started with the project, follow these steps:

1. **Clone the repository**:
   ```
   git clone https://github.com/yourusername/dr-santhosh-murthy-portfolio.git
   ```

2. **Navigate to the project directory**:
   ```
   cd dr-santhosh-murthy-portfolio
   ```

3. **Install dependencies**:
   ```
   npm install
   ```

4. **Run the development server**:
   ```
   npm run dev
   ```

5. **Open your browser** and navigate to `http://localhost:3000` to view the portfolio.
